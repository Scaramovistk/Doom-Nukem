S7 audio assets live here. The project includes simple WAV placeholders so the
sound backend works without extra downloads; replace them with final assets when
you have custom sounds.

Recognized event names:
- `door.wav`, `door.ogg`, or `door.mp3`
- `pickup.wav`, `pickup.ogg`, or `pickup.mp3`
- `shoot.wav`, `shoot.ogg`, or `shoot.mp3`
- `switch.wav`, `switch.ogg`, or `switch.mp3`
- `mission_complete.wav`, `mission_complete.ogg`, or `mission_complete.mp3`
- `mission_failed.wav`, `mission_failed.ogg`, or `mission_failed.mp3`

Background music:
- `music.wav`, `music.ogg`, or `music.mp3`

The game silently skips missing files, so maps remain playable if any asset is
removed.
